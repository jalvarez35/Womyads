Este es el repositorio para la prueba técnica womyads.

La aplicación contiene dos modelos, uno para el registro que aparece en la página inicial, y el otro para el formulario de contacto. Se incluye una página estática con gráficas de demostración de jqPlot tomadas de http://www.jqplot.com/

No se utilizaron frameworks css, no se incluye persistencia de datos. El framework utilizado es el .Net framework 4.5 en una aplicación Mvc.

Es recomendable ver la aplicación utilizando el navegador Google Chrome.
